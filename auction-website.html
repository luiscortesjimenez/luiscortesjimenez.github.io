Online Auction Platform

Python, CGI, MySQL, HTML/CSS — Group Project

A full stack web application where users register, log in, browse live auctions, place bids, and list items for sale. Built as a group project for Introduction to Software Development and deployed on university web hosting.

Highlights

  • User accounts: Registration and login with bcrypt-hashed passwords and server-side session management via HttpOnly cookies.
  • Live auctions: Browse all active listings; authenticated users can place bids with full validation (auction active, bid exceeds current high bid, no self-bidding).
  • Sell items: List an item with a name, description, starting bid, and scheduled start time; auctions run for a fixed 7-day window.
  • My listings: View your own active and past auctions.
  • Transaction history: Dashboard with views for current bids, selling activity, purchases, and expired auctions.
